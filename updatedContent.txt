## TECHNICAL NOTES

### Navigation Behavior
- Fixed position at top
- Semi-transparent white background with blur
- Responsive across all desktop sizes
- Z-index: 1000

### Form Handling
Newsletter form on get-involved.html uses POST method but action="#" (needs backend integration)

### Links
All internal navigation uses relative paths (./filename.html)
External links and form actions are placeholders (#)

### Accessibility
- Alt text on images
- ARIA labels on social media links
- Semantic HTML structure
- Focus states on interactive elements
